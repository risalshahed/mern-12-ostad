# Files Upload with Express JS & Multer

## Clone the Repository

```bash
git clone https://github.com/risalshahed/mern-12-ostad.git
```

## Change the Directory

```bash
cd mern-12-ostad
```

## Switch to the Branch

### Either with

```bash
git switch module-14-file-upload
```

### OR

```bash
git checkout module-14-file-upload
```

## Install & Run the Project

```bash
npm i
npm run dev
```
